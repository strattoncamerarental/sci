Weaver Steadman Multi-axis Jib

Includes a custom designed support base with scaffold leveling jacks to allow set up on any terrain. Also compatible with the Fisher Dolly center mount.

System Components:
  • 1: Main Jib, Secondary arm and weights
  • 2: Custom base with leveling jacks
  • 3: Weaver Steadman PAN & TILT nodal point head
  • 4: 3rd axis module for ROLL capability

For all configurations see the Weaver Steadman complete system guides ➞

Requires standard size cargo van for transport.
